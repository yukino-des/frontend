### keymap
| keymap                                      | shortcut             |  
|---------------------------------------------|----------------------|  
| delete line                                 | shift + command + K  |  
| select all occurrences of current selection | shift + command + L  |  
| find                                        | command + F          |  
| replace                                     | option + command + F |  
| format document                             | shift + option + F   |
